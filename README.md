# 🎓 Student Management System (Frontend Project)

## 📌 Overview
This is a simple and responsive Student Management System built using HTML, CSS, and JavaScript.  
It allows users to view and filter student details dynamically.

---

## 🚀 Features
- 🔍 Live search (by Name, Roll Number, Branch)
- 🧾 Dynamic student cards using JavaScript
- ❌ "No student found" error handling
- 🎨 Responsive UI with modern styling
- 📜 Scrollable container for large data

---

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript (DOM Manipulation, Array Methods)


---

## ⚙️ How It Works

1. Student data is stored in a JavaScript array.
2. Cards are dynamically created using `forEach()`.
3. Search input uses `filter()` to match:
   - Name
   - Roll Number
   - Branch
4. UI updates instantly on typing.

---

## 💡 Key Concepts Used
- DOM Manipulation
- Event Handling (`input` event)
- Array Methods:
  - `forEach()`
  - `filter()`
- Template literals
- Conditional rendering

---

## 📸 Output
Displays student cards in a grid layout with scroll support.
